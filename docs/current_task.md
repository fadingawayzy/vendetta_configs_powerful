# Current Task

## Задача
Сделать отдельный однократный запуск pipeline без запуска Telegram polling.

## Что нужно
1. Отдельный файл запуска, например run_pipeline_once.py
2. Он должен запускать pipeline один раз
3. Он не должен поднимать polling бота
4. Он не должен ломать текущий основной запуск

## Критерий готовности
- можно выполнить одну команду для разового прогона
- в логах видно этапы запуска
- основной run.py продолжает работать как раньше
